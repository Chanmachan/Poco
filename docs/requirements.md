# Poco - 要件定義書

**バージョン:** 1.0  
**作成日:** 2026-03-14  
**ステータス:** 確定

---

## 1. プロダクト概要

### 1.1 プロダクト名
**Poco**

### 1.2 目的
Macのデスクトップに付箋を貼るように使える、軽量なメニューバー常駐型TODOメモアプリ。
「思いついたらすぐメモ → デスクトップで確認 → ワンクリックで完了」というワークフローを実現する。

### 1.3 背景・課題
- 既存のリマインダーアプリは起動・入力に手間がかかる
- メモアプリは「後で見る」ことが前提で、常時視界に入らない
- 付箋のような感覚でタスクを管理できるシンプルなツールが欲しい

### 1.4 ターゲットユーザー
- 開発者・クリエイター等、Macを常用するユーザー
- 複雑なタスク管理より「メモして完了」スタイルを好むユーザー

---

## 2. スコープ

### 2.1 対象プラットフォーム
- macOS 13 Ventura 以降

### 2.2 スコープ内（v1.0）
- グローバルショートカットによるクイック入力
- デスクトップへの付箋表示（常時・ドラッグ移動・色変更）
- ワンクリック完了 + フェードアウト
- アーカイブ機能（完了済み一覧・未完了に戻す）
- メニューバー常駐

### 2.3 スコープ外（v1.0では実装しない）
- iCloud / ネットワーク同期
- リマインダー・通知機能
- タグ・カテゴリ・優先度
- iPhone / iPad 対応
- 複数ユーザー対応

## 3. 機能要件

### FR-01: グローバルショートカット
- システム全体で有効なグローバルホットキーを登録する
- デフォルトキー: `⌃ + ⌥ + N`（Control + Option + N）
- ショートカット押下で入力モーダルが最前面に表示される
- 設定画面からカスタマイズ可能（将来対応・v1.0はデフォルト固定も可）

### FR-02: クイック入力モーダル
- テキスト入力欄のみのシンプルなUI
- Enter キーで即保存 → モーダルを閉じる
- Escape キーでキャンセル（保存しない）
- 空テキストの場合は保存しない

### FR-03: デスクトップ付箋表示
- 保存されたメモをデスクトップ上に付箋として常時表示する
- ウィンドウレベル: デスクトップより前・通常ウィンドウより後ろ（最前面固定しない）
- 付箋サイズ: 220 × 140 pt（固定）
- ドラッグで任意の位置に移動可能、位置情報は永続保存される
- 付箋の色変更: 右クリックメニューから5色を選択可能

### FR-04: 完了操作
- 付箋上のチェックボタン（✓）をクリックで完了
- 完了時にフェードアウトアニメーション（約0.3秒）
- 完了後はデスクトップから非表示になる
- アーカイブに完了日時とともに保存される

### FR-05: メニューバー常駐
- アプリ起動時からメニューバーにアイコンを常駐
- Dockには表示しない（LSUIElement = YES）
- 右クリックメニュー（コンテキストメニュー）の構成:
  - アーカイブを開く
  - 環境設定（将来対応）
  - 終了

### FR-06: アーカイブウィンドウ
- 完了済みメモの一覧を表示するウィンドウ
- 表示項目: メモ内容 / 完了日時
- 各行に「未完了に戻す」ボタンを設置
  - クリックでステータスをアクティブに変更 → デスクトップに再表示

---

## 4. 非機能要件

### NFR-01: パフォーマンス
- ショートカット押下からモーダル表示まで 100ms 以内
- アプリのメモリ使用量: 平常時 50MB 以下

### NFR-02: データ永続性
- データはローカルに保存する（Core Data）
- アプリ終了・再起動後もデータ・付箋位置が復元される
- バックアップ: TimeMachineで自動的にバックアップ対象に含まれる

### NFR-03: 互換性
- macOS 13 Ventura 以降をサポート

---

## 5. 制約・前提条件

- Apple Developer Program への加入は不要（ローカル配布・開発用途）
- iCloud同期は実装しない
- Xcodeを使用したネイティブSwift/SwiftUIアプリとして実装する

---

## 6. 用語定義

| 用語 | 定義 |
|------|------|
| 付箋 | デスクトップ上に表示されるアクティブなメモウィジェット |
| アーカイブ | 完了済みメモの保管領域 |
| クイック入力モーダル | ショートカットで呼び出す入力専用のポップアップ |
