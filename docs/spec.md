# Poco - 機能仕様書

**バージョン:** 1.0  
**作成日:** 2026-03-14  
**ステータス:** 確定

---

## 1. 技術スタック

| 項目 | 選定 | 理由 |
|------|------|------|
| 言語 | Swift 5.9+ | macOSネイティブ、パフォーマンス |
| UI | SwiftUI | 宣言的UI、アニメーション実装が容易 |
| データ | Core Data | ローカル永続化、Appleの標準ORM |
| ショートカット | Carbon HotKey / CGEventTap | グローバルショートカット登録 |
| 最小OS | macOS 13 Ventura | SwiftUI機能の安定版 |

---

## 2. アーキテクチャ

```
Poco.app
├── AppDelegate              # LSUIElement設定、メニューバーアイコン管理
├── GlobalShortcutManager    # グローバルショートカット登録・検知
├── QuickInputWindow         # クイック入力モーダル
├── StickyNoteWindow         # 付箋ウィンドウ（メモ1件につき1ウィンドウ）
├── ArchiveWindow            # アーカイブ一覧ウィンドウ
├── Core Data Stack
│   ├── MemoEntity           # メモデータモデル
│   └── PersistenceController
└── StatusBarController      # メニューバーアイコン・メニュー管理
```

---

## 3. データモデル

### MemoEntity（Core Data）

| フィールド名 | 型 | 説明 |
|------------|-----|------|
| id | UUID | 一意識別子 |
| content | String | メモ本文 |
| status | String | "active" / "archived" |
| color | String | カラーコード（例: "#FFEB3B"） |
| positionX | Double | デスクトップX座標 |
| positionY | Double | デスクトップY座標 |
| createdAt | Date | 作成日時 |
| completedAt | Date? | 完了日時（完了時のみ） |

## 4. 画面仕様

### 4.1 クイック入力モーダル

```
┌─────────────────────────────┐
│  💡 新しいメモ               │  ← タイトルバーなし、角丸
│  ┌───────────────────────┐  │
│  │ ここにメモを入力...    │  │  ← テキストフィールド（フォーカス自動）
│  └───────────────────────┘  │
│  Enter: 保存  Esc: キャンセル │  ← ヒントテキスト（小さく表示）
└─────────────────────────────┘
  幅: 320pt  高さ: 90pt
  表示位置: 画面中央上部（y: 200pt固定）
```

- 表示時: スライドイン（上から）アニメーション
- テキストフィールドに自動フォーカス
- 空テキストでEnterは無効

### 4.2 デスクトップ付箋

```
┌────────────────────────┐
│ ✓  [完了ボタン]         │  ← ヘッダー部分（カラーに応じて配色）
├────────────────────────┤
│                        │
│  メモ本文がここに       │  ← テキスト（折り返しあり）
│  表示される            │
│                        │
└────────────────────────┘
  幅: 220pt  高さ: 140pt  角丸: 8pt
```

**インタラクション:**
- ドラッグ: ウィンドウ全体をドラッグして移動
- 右クリック: コンテキストメニュー表示
  - 色変更: 🟡 黄 / 🔵 青 / 🟢 緑 / 🩷 ピンク / ⬜ 白
  - 削除（確認ダイアログあり）
- ✓ ボタンクリック: 完了処理

**カラー定義:**

| 名称 | 背景色 | ヘッダー色 |
|------|--------|----------|
| 黄（デフォルト） | #FFF9C4 | #F9A825 |
| 青 | #E3F2FD | #1565C0 |
| 緑 | #E8F5E9 | #2E7D32 |
| ピンク | #FCE4EC | #C62828 |
| 白 | #FAFAFA | #9E9E9E |

### 4.3 アーカイブウィンドウ

```
┌─────────────────────────────────────────┐
│  Poco - アーカイブ                   [×] │
├─────────────────────────────────────────┤
│  完了済みメモ (3件)                      │
├──────────────────────────┬──────────────┤
│ メモ内容                  │ 完了日時      │
├──────────────────────────┼──────────────┤
│ 牛乳を買う                │ 3/14 11:00   │ [未完了に戻す]
│ PRレビューする             │ 3/13 18:30   │ [未完了に戻す]
│ 請求書を送る              │ 3/12 09:15   │ [未完了に戻す]
└──────────────────────────┴──────────────┘
  幅: 500pt  高さ: 400pt
```

- 完了日時の降順で表示
- 「未完了に戻す」クリックでアクティブに戻り、デスクトップに再表示

## 5. 状態遷移

```
[作成] → active（デスクトップ表示）
                ↓ ✓ クリック
           フェードアウト（0.3秒）
                ↓
          archived（アーカイブ保存）
                ↓ 未完了に戻す
           active（デスクトップ再表示）
```

## 6. アプリ起動フロー

1. `applicationDidFinishLaunching`
2. `LSUIElement = YES` → Dockに表示しない
3. メニューバーアイコンをセット（SF Symbol: `note.text`）
4. Core Dataスタック初期化
5. グローバルショートカット登録（`⌃ + ⌥ + N`）
6. activeステータスのメモを全件読み込み → 付箋ウィンドウを生成・表示
7. 待機状態

---

## 7. グローバルショートカット実装方針

- `CGEventTap` を使用してシステム全体のキーイベントを監視
- キー組み合わせ: `kCGEventFlagMaskControl | kCGEventFlagMaskAlternate` + `n`
- Accessibility権限が必要（初回起動時にシステム環境設定へ誘導）

---

## 8. 付箋ウィンドウ実装方針

- `NSWindow` を `NSWindowLevel` = `.desktopIcon + 1` で生成
- `styleMask = .borderless`（タイトルバーなし）
- `isMovableByWindowBackground = true`（ドラッグ移動）
- `backgroundColor = .clear`（SwiftUIのShapeで見た目を実現）
- Spaces/Mission Control をまたいで表示: `collectionBehavior = [.canJoinAllSpaces, .stationary]`
